# 神港学園 オープンハイスクール・入試説明会 事後アンケート集計結果

このサイトは、神港学園のオープンハイスクール・入試説明会の事後アンケート結果をわかりやすくまとめたものです。

## 概要

教職員によるアンケート結果を、以下の4つのセッションに分けて表示しています：

- **セッション1：実施形式について**
  - 開催時期・回数の適切性
  - 情報発信（集客・告知）の方法

- **セッション2：内容・プログラムについて**
  - オープンスクール（体験・見学型）の内容評価
  - 入試説明会（情報伝達型）の内容評価

- **セッション3：運営体制について**
  - 役割分担と負担感

- **セッション4：自由記述**
  - 神港学園の「強み」をさらに伝えるための提案

## ファイル構成

```
.
├── index.html      # メインのHTMLファイル
├── styles.css      # スタイルシート
├── script.js       # JavaScript（グラフ表示・データ処理）
├── data.js         # アンケートデータ（JSON形式）
└── README.md       # このファイル
```

## データの更新方法

アンケートデータを更新する場合は、`data.js` ファイルを編集してください。

データ構造は以下の通りです：

```javascript
const surveyData = {
    session1: {
        question1: {
            title: "質問タイトル",
            responses: {
                "回答1": 数値,
                "回答2": 数値
            },
            improvements: [
                "改善点1",
                "改善点2"
            ]
        }
    }
    // ...
};
```

## ローカルでの確認方法

1. このリポジトリをクローンまたはダウンロード
2. `index.html` をブラウザで開く

または、ローカルサーバーを起動：

```bash
# Python 3の場合
python -m http.server 8000

# Node.jsの場合（http-serverが必要）
npx http-server
```

その後、ブラウザで `http://localhost:8000` にアクセス

## GitHub Pagesでの公開

1. このリポジトリをGitHubにプッシュ
2. リポジトリの Settings > Pages に移動
3. Source を "main" ブランチ（または使用しているブランチ）に設定
4. 数分後に `https://[ユーザー名].github.io/[リポジトリ名]` でアクセス可能になります
